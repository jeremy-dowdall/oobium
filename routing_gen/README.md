# oobium_routing_gen
[![pub package](https://img.shields.io/pub/v/oobium_routing.svg)](https://pub.dev/packages/oobium_routing)

Generators for Oobium Routing

# Usage
To use this plugin, add `oobium_routing_gen` as a [dev_dependency in your pubspec.yaml file](https://flutter.dev/platform-plugins/).

```shell script
# dart
> dart run build_runner build

# flutter
> flutter pub run build_runner build
```
